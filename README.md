# ResumeAI
All-in-one AI resume platform with LinkedIn &amp; GitHub analysis, AI chat assistance, intelligent job matching, and customizable templates to build high-impact resumes.
## ✨ Features

- 🔗 One-click LinkedIn profile analysis  
- 💻 GitHub profile evaluation  
- 🤖 AI chat assistant
-   resume improvement  
- 🎯 Smart job matching system  
- 📄 Customizable resume templates  
- 📊 ATS score calculation  
- ⚡ Resume parsing & structured data extraction  

---

### Frontend
